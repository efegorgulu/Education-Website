# Software Education Website (HTML & CSS)

A static **software academy promotional website** developed using **HTML5 and CSS3**, presenting course offerings, global campuses, education programs, student testimonials, and contact-focused call-to-action sections within a clean and structured modern layout.

## Screenshot
![](Screen.gif)

## Highlights
- **Hero banner section** with academy introduction and primary action button  
- **Responsive-style navigation layout** with menu toggle and branded logo  
- **Courses section** presenting key learning paths such as Web, Mobile, and Backend Development  
- **Global campus showcase** with layered image overlays for location highlights  
- **Education program cards** featuring frontend, backend, and mobile development tracks  
- **Student testimonials section** with profile images, reviews, and rating visuals  
- **Call-to-action area** encouraging user engagement and contact  
- **Footer section** containing academy information, social media icons, and copyright notice  

## Technologies Used
- **HTML5**
- **CSS3**
- **Font Awesome Icons** for visual interface elements  

## Installation
1. Clone or download this repository  
2. Open `index.html` in your preferred web browser  

## Project Notes
- Front-end presentation only (**no backend or dynamic JavaScript functionality**)  
- Designed for **layout structuring, styling practice, and educational portfolio use**  
- Images and visual assets are stored locally within the project directories
